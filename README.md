## CAP 原则
- CAP原则又称CAP定理，指的是在一个分布式系统中，一致性（Consistency）、可用性（Availability）、分区容错性（Partition tolerance）
- CAP 原则指的是，这三个要素最多只能同时实现两点，不可能三者兼顾
- CP：高一致性
- AP：高可用性
